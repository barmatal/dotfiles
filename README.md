# Instalation

- Clone repo
- Create simlinks for each file

**On Windows**

From `$HOME`:

`mklink .\_vimrc .\projects\dotfiles\vimrc`

**On Mac**

`ln -s ~/projects/dotfiles/vimrc ~/.vimrc`

# List of files

- **vimrc**: vim dot file. Rename/link using `.vimrc` for Mac/Linux and `_vimrc` for windows
